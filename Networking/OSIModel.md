# OSI Model

## 7 Layers
1. Physical
2. Data Link
3. Network
4. Transport
5. Session
6. Presentation
7. Application


* Application, Presentation and Session layers belong to Application Layer.


### Application Layer:


* Application Layer has many protocols including 
  HTTP, SMTP, FTP, SSH, DNS

### Transport Layer:

* Transport layer protocols provide end-to-end communication
  services for applications. TCP and UDP are primarily used for communication.
